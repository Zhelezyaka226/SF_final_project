


# Прогнозирование стоимости недвижимости

Этот проект реализует модель машинного обучения для прогнозирования стоимости недвижимости и предоставляет веб-сервис на базе Flask для обработки запросов с данными о недвижимости.

## Описание

Приложение позволяет:
- Обучить модель на основе данных о недвижимости (количество спален, ванных комнат, площадь и т.д.).
- Экспортировать модель в файл формата `.joblib`.
- Развернуть веб-сервис, который принимает данные о недвижимости в формате JSON и возвращает прогнозируемую стоимость.

## Технологии

- Python 3.8+
- Flask
- scikit-learn
- joblib

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/username/real_estate_prediction.git
   cd real_estate_prediction
	
2. Установите зависимости:

bash

pip install -r requirements.txt


3. Убедитесь, что данные в формате CSV находятся в папке проекта.
Использование

4. Обучение модели
Обучите модель на основе данных о недвижимости:
bash

python train_model.py
Скрипт train_model.py обучит модель и сохранит её в файл real_estate_model.joblib.

5. Запуск веб-сервиса
Запустите Flask-сервис:
bash

python app.py
Сервис будет доступен по адресу: http://127.0.0.1:5000

6. Пример запроса к API
Отправьте POST-запрос с данными о недвижимости:
bash

curl -X POST -H "Content-Type: application/json" -d '{
    "baths": 2,
    "beds": 3,
    "sqft": 1500,
    "stories": 2
}' http://127.0.0.1:5000/predict
Ответ:
json


{
    "predicted_price": 350000.0
}

Структура проекта
	•	train_model.py: скрипт для обучения и сохранения модели.
	•	app.py: файл веб-сервиса на Flask.
	•	real_estate_model.joblib: файл с обученной моделью (создается после выполнения train_model.py).
	•	requirements.txt: список необходимых библиотек.
	•	README.md: документация проекта.
